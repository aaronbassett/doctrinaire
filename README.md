# doctrinaire

An opinionated Vite & React starter template for building web apps with TypeScript.

## Included

- [Vite](https://vitejs.dev/)
- [React](https://reactjs.org/)


## Getting Started

```bash
git clone https://github.com/aaronbassett/doctrinaire.git
cd doctrinaire
bun install
bun run dev
```
